# AI Skills for LAN Share

Local markdown copies of relevant skills found from skills.sh for this C++/Qt project.

## Recommended for this repo

- `make.SKILL.md` - Most relevant installed skill for C/C++ build-system work. LAN Share uses Qt/qmake and Windows nmake in the packaging helpers, so Makefile concepts apply most directly when debugging generated makefiles, incremental builds, compiler/linker flags, and build targets.

## Useful but not exact

- `building-qt-apps.SKILL.md` - Qt app guidance, but mainly written for Python/PySide6 and qasync. Some event-loop and signal/slot cautions are conceptually useful, but do not treat the Python architecture examples as direct C++/Qt Widgets guidance.
